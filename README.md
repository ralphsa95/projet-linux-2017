# projet-linux-2017

Il faut compiler -> saca.c: gcc saca.c -o saca -lpthread
                 -> avion.c: gcc avion.c -o avion -lpthread -lm
                 -> control.c: gcc control.c -o control
Ensuite, il faut déclencer saca premièrenent : ./saca
puis control et avion: ./control
                       ./avion
